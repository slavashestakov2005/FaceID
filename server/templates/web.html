Съёмка видео

На этой странице вы можете снять своё видео с web камеры, чтобы потом мы нашли на нём Ваше лицо.

Для начала записи нажмите на кнопку «Начать запись».

После нажатия на кнопку «Останосить запись» Ваше видео начнёт загружаться на сайт, и, как только на нём будут найдены все лица, мы переметим тебя на следующую страницу.
